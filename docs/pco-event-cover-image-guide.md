# Event Cover Images — Staff Guide

Every public event you create in Planning Center Calendar should have a cover image. The Mosaic website pulls these directly from PCO every two hours — no manual website update needed. Once you upload an image to an event in PCO, it shows up on the site automatically.

This guide covers what to upload, where to upload it, and a few quick rules of thumb for picking a good image.

---

## Why this matters

The website's homepage features three upcoming events as visual cards. Without a cover image, the card falls back to a plain blue gradient with the event name — readable, but missing the visual punch that draws first-time visitors in. **An event without a cover image is essentially invisible** on the homepage compared to one with a great photo or graphic.

This is also true of the All Events listing page (when shown in grid view) and the individual event detail pages.

---

## What makes a good cover image

- **Wide, not square.** The website uses landscape (16:9) crops. Vertical or square images get cut off awkwardly.
- **Minimum 1200×675 pixels.** Larger is better — the website resizes down. Images smaller than this look fuzzy on retina screens.
- **Important content in the middle.** The card sometimes overlays text on the bottom-left or shows the image partially blurred at the right edge — keep your subject (face, key text) in the central 60% of the image.
- **High contrast, simple composition.** Cards are small. A single subject or piece of text reads better than a busy collage.
- **Match the vibe of the event.** A retreat poster, a Bible study graphic, a candid group photo — pick the medium that fits. You don't need a designed graphic for every event; a good photo is often better.

**Quick examples:**
- Retreats / weekends → designed poster (e.g., "Soda Summer")
- Bible studies / classes → simple graphic with a key word or icon
- Recurring meetings (prayer, communion, board meetings) → candid photo from a previous occurrence, or just a stock-style image of the topic

## Step-by-step

### 1. Open the event in Planning Center Calendar

Go to **https://calendar.planningcenteronline.com/** and sign in.

Find the event in your event list. If it's a recurring event (like Bible Study), click the parent event — uploading a cover image to the parent automatically applies it to every instance.

---

### 2. Open the event details

Click the event name to open its details page.

---

### 3. Upload the cover image

Look for the **"Image"** field in the event settings. Click **"Choose Image"** (or the camera icon — Planning Center updates this UI occasionally).

In the upload dialog:
1. Click **"Upload"** or drag your image file in.
2. Wait for the upload to finish — usually under 5 seconds.
3. Crop the image if prompted. Aim for a **landscape (16:9)** crop with the main subject centered.
4. Click **"Save"** or **"Done"**.

---

### 4. Make sure the event is visible in Church Center

While you're in event settings, find the **"Visibility"** section and confirm **"Visible on Church Center"** is enabled. The website **only shows events that are visible on Church Center** — internal-only events (staff meetings, etc.) are correctly excluded.

---

### 5. Save and you're done

Click **Save** at the bottom of the event page. The website will pick up the new image within 2 hours (next cron run). If you need it live immediately, message Dave to trigger a manual sync.

## Common questions

**"Do I need to upload an image for every recurring instance?"**
No — upload once to the parent event, and all instances inherit it. PCO Calendar handles this automatically.

**"What if I don't have a good image yet?"**
Better to publish the event without an image than delay creating it — the gradient fallback still shows the event name and links to the detail page. You can come back and add an image later, and the site picks it up on the next sync.

**"Where can I find free, good-looking event images?"**
- Unsplash (https://unsplash.com/) — free photography, no attribution required
- Pexels (https://pexels.com/) — same idea, different photo library
- Lightstock (https://lightstock.com/) — Christian-specific photography (not free, but high quality)
- For graphic-style cards: Canva (https://canva.com/) has church event templates

**"Can I use a photo with people from a previous event?"**
Yes, as long as you have permission. Avoid photos where individuals are clearly identifiable unless they've signed a media release. Group/crowd shots from worship or community gatherings are usually safe.

## What's automated, what's not

**Automated (no action needed):**
- Pulling events from PCO every 2 hours
- Filtering out non-public events
- Resizing images for different devices
- Updating the website when you change an event in PCO

**Still manual:**
- Uploading the cover image (this guide)
- Setting "Visible on Church Center" on each event
- Filling in a clear event name and short summary
